Шаги для запуска
1. Импортируйте коллекции в Postman
- Откройте Postman.
- Нажмите кнопку Import (в левом верхнем углу).
- Выберите вкладку File и загрузите файлы:
- API auth.postman_collection.json
- API carts.postman_collection.json
- Нажмите Import.
- После импорта коллекции появятся в левой панели (в разделе «Collections»).

2. Запуск через Postman Runner (для всех запросов сразу)
- Нажмите на название коллекции (например, API auth) и выберите Run collection.
- В окне Runner:
- Убедитесь, что выбрана нужная коллекция.
- При необходимости установите задержку между запросами (например, 200ms), чтобы избежать ошибок ECONNRESET.
- Нажмите Run API auth.
- Аналогично можно запустить коллекцию API carts.

3. Ожидаемые результаты
- Все тесты должны пройти успешно (зелёные индикаторы в отчете) !КРОМЕ! ЗАПРОСА NEGATIV  https://dummyjson.com/carts/9999 он должен упассть с 404 ошибкой, но тесты должы быть пройдены успешно.
- Токены автоматически сохраняются и используются в последующих запросах.
- При ошибке проверьте наличие интернет-соединения и работоспособность сервиса dummyjson.com.

4. Иногда запросы падают с ошибкой read ECONNRESET. Возможные причины
- Сервер превысил лимит соединений или закрыл соединение преждевременно
- Таймауты на клиентской стороне
- Ошибки в самом серверном приложении
